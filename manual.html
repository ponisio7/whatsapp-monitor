🪟 Instalación en Windows

1
Instalar Python

Descarga Python 3.12 desde python.org

⚠️ IMPORTANTE: Marca ✅ "Add Python to PATH" durante la instalación

Verifica la instalación:

python --version

Deberías ver: Python 3.12.x

2
Descargar el proyecto

Crea una carpeta para el proyecto:

mkdir C:\WhatsAppMonitor
cd C:\WhatsAppMonitor
3
Copiar los archivos del proyecto

Copia todos los archivos .py, AGENT.md y requirements.txt a esta carpeta.

4
Instalar dependencias

Abre Símbolo del sistema (CMD) como administrador:

cd C:\WhatsAppMonitor
python -m pip install --upgrade pip
python -m pip install -r requirements.txt

Si hay errores, instala uno por uno:

python -m pip install playwright openai python-dotenv PyQt6 numpy pillow
python -m pip install opencv-python pytesseract
5
Instalar navegador Playwright
python -m playwright install chromium
6
Configurar API Key de DeepSeek

1. Obtén una API Key en platform.deepseek.com

2. Crea un archivo .env en la carpeta del proyecto:

echo DEEPSEEK_API_KEY=tu_api_key_aqui > .env
7
Ejecutar la aplicación
python launcher.py

🐧 Instalación en Linux (Ubuntu/Debian)

1
Verificar Python instalado
python3 --version

Versiones aceptables: 3.9, 3.10, 3.11, 3.12

✅ Si ya tienes Python 3.9-3.12, no necesitas instalarlo, pasa al paso 2.
⚠️ Python 3.12 podrías tener problemas con algunas librerías. Recomendamos usar 3.11.
2
Instalar Python SOLO si es necesario

Solo ejecuta esto si NO tienes Python o tu versión es inferior a 3.9

sudo apt update
sudo apt install -y python3.11 python3-pip python3-venv
3
Verificar pip (gestor de paquetes)
pip3 --version

Si no está instalado:

sudo apt install -y python3-pip
4
Instalar dependencias del sistema (necesarias para Playwright)
sudo apt install -y \
libnss3 libatk-bridge2.0-0 libdrm2 libxkbcommon0 \
libgbm1 libasound2 libxshmfence1 libgtk-3-0 \
libxcomposite1 libxcursor1 libxdamage1 libxrandr2 \
libpango-1.0-0 libcairo2 libatspi2.0-0 libxfixes3

Dependencias para OCR:

sudo apt install -y tesseract-ocr tesseract-ocr-spa
5
Crear carpeta del proyecto
mkdir -p ~/WhatsAppMonitor
cd ~/WhatsAppMonitor
6
Copiar los archivos del proyecto

Copia todos los archivos .py, AGENT.md y requirements.txt a esta carpeta.

7
Crear entorno virtual (recomendado)
python3 -m venv venv
source venv/bin/activate
💡 El entorno virtual aísla las dependencias y evita conflictos con otros proyectos.
8
Instalar dependencias Python
pip install --upgrade pip
pip install -r requirements.txt
9
Instalar Playwright
python -m playwright install chromium
10
Configurar API Key
echo "DEEPSEEK_API_KEY=tu_api_key_aqui" > .env
💡se explica en la parte de WINDOWS, donde obtener la clave (KEY) :
6
Configurar API Key de DeepSeek
11
Ejecutar la aplicación

Si usas entorno virtual:

source ~/WhatsAppMonitor/venv/bin/activate
python launcher.py

Si NO usas entorno virtual:

python3 main.py

🔧 Solución de problemas comunes

❌ Error: uvloop no funciona en Windows

La aplicación ya incluye una detección automática. Si ves errores, edita main.py y comenta las líneas de uvloop.

❌ Error: Playwright no encuentra chromium
python -m playwright install chromium --force
❌ Error: DEEPSEEK_API_KEY not found

Verifica que el archivo .env existe y tiene el formato correcto:

DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxx
❌ Error: PyQt6 no se instala en Linux
sudo apt install -y python3-pyqt6
pip install PyQt6 --no-cache-dir
⚠️ El navegador no se abre o se cierra solo
  • Cierra WhatsApp Web en tu navegador normal
  • Borra la carpeta session_wa_jobs y vuelve a intentar
  • Verifica que no haya antivirus bloqueando

🚀 Primer uso

  1. Ejecuta python launcher.py
  2. Configura tu API Key de DeepSeek (Botón "🔑 CONFIGURAR API")
  3. Presiona "▶ INICIAR"
  4. Escanea el código QR con tu WhatsApp
  5. ¡El monitor comenzará a funcionar!
